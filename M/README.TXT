----------------------------------------------------------------
* M File Manager

  90년대 MS-DOS 용 프로그램인 Mdir과 NC(Norton Commander)에서
   영감을 얻어서 만든 MSX-DOS2 용 파일관리자입니다.

  메모리 매퍼의 남은 용량이 224KB 이상이면 일반모드로 실행되며,
  남은 용량이 128KB ~ 192KB 사이에서는 Lite모드로 실행됩니다.
  Lite모드에서는 아래의 일부 기능이 정지됩니다.
  인터레이스 모드, 한글처리, LFN 표시, IMS 플레이어, MPX F/W.
  만약 남은 용량이 112KB 이하인 경우는 M이 실행되지 않습니다.

  파일 및 디렉토리를 복사, 이동, 삭제, 속성변경 등을 지원합니다.
  복사/이동 시 원본(source) 디렉토리는 커서가 보이는 패널이며,
  목적(target) 디렉토리는 커서가 없는 다른쪽 패널이 됩니다.
  (예: 왼쪽 패널의 파일에 커서를 놓고 복사 명령을 내리면,
       해당 파일은 오른쪽 패널의 디렉토리로 복사됩니다.)

  다양한 프로그램 런처기능을 가집니다.
  펑션키 또는 확장자별 실행프로그램을 선택할 수 있습니다.
  자세한 내용은 "* M.INI 설정" 부분을 참고하세요.

  인터레이스 모드에서는 완성형(KS-5601) 한글 출력이 가능하며,
   넌인터레이스 모드에서는 아스키 문자만 출력됩니다.
  특수문자 및 한자표시를 위해 램상주 폰트를 쓸 수 있습니다.
  (예: KS_SPE.COM, KS_HANJA.COM)

  한글 FontPack 카트리지를 사용하면,
   확장완성형 한글(CP949) 및 한자 표시도 가능합니다.

  VFAT의 LFN(긴 파일명)을 표시할 수 있습니다.
  유니코드 처리를 위해 FontPack V1/V2 카트리지가 필요합니다.
  영문(U+0000 ~ U+00FF) 처리는 FontPack이 없어도 가능합니다.
  
  내장된 사운드, 그래픽 관련 기능을 활용가능합니다.
  (예: IMS Player, MP3/WMA Player, Graphic Viewer)
  음악 재생 시, ID3v1, ID3v2, WMA, GD3, PT2/3 태그를 표시합니다.
  (단 ID3v2 처리 시, FontPack 필수)

  WAU 리모트 드라이브의 모듈 음악(MOD, S3M, XM 등) 스트리밍 시,
  제목 태그를 표시합니다.

  WAU Tiny Browser가 들어있습니다.
  GitHub, MSX-Archive, File-Hunter 사이트를 지원합니다.


----------------------------------------------------------------
* 요구 사항

  공통: MSX2, MSX-DOS2, VRAM 128KB

  일반모드: 메인 메모리매퍼 224KB 사용
            모든 기능을 사용가능

  Lite모드: 메인 메모리매퍼 128KB 사용
  
  주의: 보조 메모리매퍼를 지원하지 않습니다.
        따라서 FS-A1ST에 메모리매퍼 카트리지를 장착하더라도,
        M은 메모리 부족 에러를 표시하며 실행되지않습니다.
        이 경우, 내장 메모리를 512KB 이상 증설하면 됩니다.


* 옵션 사항

  한글 FontPack V1/V2 카트리지 (CP949, Unicode 표시)
  MPX R1.x/R2.0 및 WAU 카트리지 (MP3, WMA 오디오 플레이어)
  MoonSound/달소리 (IMS 오디오 플레이어)
  PAC-V/XII-V 카트리지 (LED 비주얼라이저)
  WAU 카트리지 (PC 리모트 드라이브, Tiny Browser)


----------------------------------------------------------------
* 설치 방법

  1. 압축파일을 M 디렉토리에 풀어넣습니다.
  2. M.INI 파일을 자신의 환경에 맞게 편집합니다.
  3. M 디렉토리를 AUTOEXEC.BAT의 PATH 설정에 추가합니다.


* 실행 방법

  DOS2 프롬프트에서 M [RETURN]을 타이핑하면 실행됩니다.

  M은 기본적으로 램에 상주한 상태로 동작합니다.
  만약 램상주를 해제하려면, DOS2 프롬프트에서 M /R를 입력하세요.
  M.INI 파일에서 TSR:0으로 놓으면 램상주 기능을 끌 수 있습니다.

  파일 뷰어 상에서의 동작키에 대한 설명은,
   M에서 #HELP를 실행하면 볼 수 있습니다. (F1키에 기본 등록됨)

  CTRL키 조합의 단축키 커맨드는,
   M에서 CTRL키를 누르면 하단에 짧게 표시되니 참고하세요.


----------------------------------------------------------------
* 단축키 목록

  키 목록의 ^는 CTRL키, #는 SHIFT키를 의미합니다.

  #ESC    : M을 종료하고 DOS2로 복귀
   ESC    : DOS2 프롬프트 화면 보기 / 명령 취소
   HOME   : Quick-Dir 메뉴

   TAB    : 두 패널간의 커서 이동
   UP     : 커서를 위 항목으로 이동
  ^UP     : 커서를 첫 항목으로 이동
   DOWN   : 커서를 아래 항목으로 이동
  ^DOWN   : 커서를 마지막 항목으로 이동
   LEFT   : 커서를 이전 페이지로 이동
   RIGHT  : 커서를 다음 페이지로 이동

   RETURN : 파일 실행 / 서브디렉토리 열기 / 드라이브 열기
  ^RETURN : 타겟 패널에서 열기 (서브 디렉토리, 드라이브)
  #RETURN : 파일 또는 디렉토리 속성 표시

  F1 ~ F5 : 펑션키 메뉴 실행 (#:F6~F10, #^:F11~F15)
  #A ~ #H : 드라이브 빠르게 열기 (A,B,C ... H)
   \      : 루트 디렉토리로 이동
  #\      : M 디렉토리로 이동
   BS     : 상위 디렉토리로 이동
   /      : 도스 커맨드라인 열기
  #/      : 도스 커맨드라인 열고 선택파일 붙여넣기

   SPACE  : 파일 선택 또는 해제 (길게 눌러 아래쪽으로 연속선택)
  #SPACE  : 모든 파일 선택 또는 해제
  #UP     : 현재 파일 선택 후 커서를 위 항목으로 이동
  #DOWN   : 현재 파일 선택 후 커서를 아래 항목으로 이동

  ^C | INS: 파일 또는 디렉토리 복사
  ^M      : 파일 또는 디렉토리 이동
  ^S      : 리스트 정렬방식 변경
  ^D | DEL: 파일 또는 디렉토리 삭제 (ReadOnly 속성 제외)
  #DEL    : 파일 또는 디렉토리 삭제 (ReadOnly 속성 포함)
  ^K      : 디렉토리 생성
  ^R      : 파일 또는 디렉토리 이름 변경

  ^B      : 'about' 윈도 표시
  ^A      : 선택된 파일을 LZH로 압축
  ^E      : LZH 파일을 압축해제
  ^I      : 인터레이스, 넌인터레이스 모드 전환
  ^N      : 흑백, 컬러 모드 전환
  ^V      : V-Sync 60Hz, 50Hz 모드 전환
  ^L      : LFN(긴파일명) 모드 전환


* 파일/디렉토리 이름 검색

  키보드로 이름을 입력하면,
   해당 이름으로 시작하는 항목으로 커서가 이동합니다.
  
  현재 입력중인 이름을 취소하려면, ESC 키를 누르세요. 
   또는 UP/DOWN/TAB 키 등으로 커서를 이동하세요.


----------------------------------------------------------------
* M.INI 설정

  사용자 환경 설정은 M.INI 파일로 저장됩니다.
  펑션키 및 파일 확장자별 실행 프로그램을 설정합니다.
  자세한 설명은 M.INI 파일 내부의 주석을 참고하세요.

  확장자에 따른 글자색(16진수 사용)을 지정할 수 있으며,
   배경색도 바꿀 수 있습니다.
  파일의 기본색은 회색(E)이며, 배경색은 파란색(BG:4)입니다.

  프로그램 설정시 %s는 선택된 파일/디렉토리명으로 대체되며,
  %f는 확장자를 제외한 이름으로 대체됩니다.
  세미콜론으로 여러개의 명령을 리스트로 쓸 수 있습니다.
  예) TXT:5:HV %s;APED %s
      MP3:9:#MPXP %s;MPXTOOL %s /v2

  기본적으로 TSR(램상주) 옵션이 켜져 있습니다.
  본체 메모리가 256KB 이하인 경우는 TSR 해제하는 편이 좋습니다.
  TSR 기능을 끄려면, <OPT> 그룹에서 TSR:0으로 설정하세요.

  CTRL + B를 눌러 M.INI의 메모리 사용량을 확인할 수 있습니다.
  M.INI의 설정 데이터는 최대 4608바이트입니다.


* 임시 파일 생성

  LZH 압축 파일 처리시 _M$.BAT 파일이 임시로 생성됩니다.
  정상적인 동작 후 삭제되니 참고하세요.


* 명령어 접두사

  *: 명령어/실행파일 구동 시,
     M이 점유중인 메모리가 해제된 후 프로그램 실행됩니다.
     (메모리 부족한 환경에서 유용함.)
     M으로 복귀 시, 다시 메모리에 상주합니다.
     예) *MEMORY

  ]: 명령어/실행파일을 Full Path로 입력하면,
     해당 디렉토리로 이동 후, 프로그램이 실행됩니다.
     M으로 복귀 시, 기존 디렉토리로 돌아옵니다.
     예) ]B:\M\APED M.INI

  /: 명령어/실행파일 구동 시,
     DOS 텍스트 모드로 전환하지 않고, M 화면을 유지합니다.
     그래픽을 쓰지 않는 프로그램 실행 시 편리합니다.
     예) /MMCSD MUSIC.WAV


* 내장 프로그램

  #HELP: 도움말 출력 (M.HLP 표시)

  #CDD : 드라이브/디렉토리 이동
         자주 사용하는 디렉토리를 펑션키에 등록하면 편합니다.
         예) #CDD B:\UTILS;#CDD C:\GAME\ROM

  #CPU0: turboR Z80 모드 변경
  #CPU1: turboR R800 ROM 모드 변경
  #CPU2: turboR R800 DRAM 모드 변경

  #WAUB: WAU 카트리지를 사용하는 Tiny Browser입니다.
         아래 사이트의 브라우징을 할 수 있습니다.
         #WAUB GitHub <account>
         #WAUB MSX-Archive

  #MPXP: MPX R1.x/R2.0 및 WAU 카트리지를 사용하는 MP3/WMA 플레이어입니다.
         화면 하단에 MP3/WMA 진행바가 표시되며,
         MP3 ID3v1/v2 태그 및 WMA info 태그를 지원합니다.
         PAC-V/XII-V LED 비주얼라이저를 지원합니다.

  #IMSP: MoonSound 카트리지를 사용하는 IMS 음악 플레이어입니다.
         (Lite 모드에서는 사용불가)
         PAC-V/XII-V LED 비주얼라이저를 지원합니다.
         turboR에서는 가사(ISS) 표시도 가능합니다.

  #VGMP: 외부 프로그램인 VGMPlay을 활용한 VGM 플레이어입니다.
         GD3 태그를 지원합니다.

  #PT3P: 외부 프로그램인 PT3Play을 활용한 PT3 플레이어입니다.
         곡정보 표시를 지원합니다.

  #GV  : 그래픽 뷰어
         GE5/7, SRn, Snn, SCn, PIC, YJK, ILn, SGX
