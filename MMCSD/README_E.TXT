[ BIOS & Tool v5.75 ]

- MMC/SD BIOS

  Added BIOS ROM in MMCSD.OVL
  -> MMCSD.COM will update BIOS automatically,
      if BIOS of device is outdated.


- SUB-ROM

  Added optional max.28 characters for title.
  -> Use '/S:' instead of '/S'
     Eg) MMCSD RC749.ROM /S:THE_MAZE_OF_GALIOUS


[ UPDATE GUIDE ]

1. Copy MMCSD.COM, MMCSD.OVL to UTILS directory.

2. Execute MMCSD.COM.

3. Press Y key after 'Update BIOS-ROM? [y/N]' message.

4. Reboot when downloading is done.

5. END
